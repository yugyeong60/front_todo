방명록

2024학년도 1학기 오픈소스소프트웨어 개인프로젝트 과제

개인프로젝트 주제 : 자기소개 페이지와 간단한 방명록을 남기고 볼 수 있는 웹페이지
자기소개 페이지 관련 세부 내용 : 자기 자신을 소개하기 위한 어떤 콘텐츠도 허용함, 인터랙티브 요소들을 적절히 활용 (오픈소스 활용가능)
방명록 관련 세부 내용: 방명록의 내용은 fastapi가 종료될 때 삭제되어도 무방함.(보존되지 않아도 됨), 방명록 입력 시 작성자와 내용을 입력 받아야 함, 방명록을 표시할 때는 작성자, 내용, 작성시간이 표시되어야 함, (optional) 각 방명록 내용에는 삭체 버튼이 존재하며, 누구나 삭제할 수 있음
